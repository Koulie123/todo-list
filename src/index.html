Welcome

This is a TODO app that works off of the local storage on your computer.
That means that it will save the todos as they are noted on this computer, but they will not be visible on any other computer.
Additionally, the form for creating a todo on mobile doesn't always display correctly.
Currently this app only works well on desktop, appoligies if the display is not great on mobile.
To get started:
  • If you would like to group your todos into projects, you can click on the button to create a project.
  • Create a new Todo with the button on the side, if you're grouping todos into projects, select the project you would like to add it to from the dropdown menu.
  • To view all todos grouped by their project, click on the "All Projects" button
